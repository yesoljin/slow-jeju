# 슬로우 제주

제주 관광지의 향후 30일 예상 집중률과 연관 관광지를 비교하여 더 여유로운 장소와 날짜 선택을 돕는 관광 분산 웹앱입니다.

## 주요 기능

- 자연·레저·체험·역사·문화 관광지 지도
- 관광지별 30일 붐빔 달력
- 연관 대체 관광지 지도 표시
- 장소 또는 날짜 변경 전후의 혼잡 부담 감소율 비교
- 날짜가 정해지지 않은 사용자를 위한 30일 평균 탐색

## 사용 데이터

- 한국관광공사 국문 관광정보 서비스
- 한국관광공사 관광지 집중률 방문자 추이 예측 정보
- 한국관광공사 관광지별 연관 관광지 정보

현재 공개 시연본의 일부 방문객 수치는 화면 검증용 샘플이며, 실제 운영 단계에서는 정기 수집 데이터로 교체합니다.

## 실행 방법

Node.js 22.13.0 이상이 필요합니다.

```bash
npm install
npm run dev
```

운영용 결과를 만들려면 다음 명령을 실행합니다.

```bash
npm run build
```

## 주요 파일

- `app/page.tsx`: 화면과 사용자 흐름
- `app/globals.css`: 브랜드 디자인과 반응형 배치
- `app/sample-data.json`: 관광지·집중률·연관지 샘플
- `standalone-template.html`: 공유용 단일 HTML 템플릿
- `scripts/generate-standalone.mjs`: 공유용 HTML 생성

API 인증키는 소스에 저장하지 않고 서버 환경변수로 관리합니다.
